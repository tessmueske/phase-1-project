# phase-1-project
final project for phase one - javascript

~~~~~~~~~~~~~~~~~~~~~~~~~

This is an interactive and informative info sheet on Mediterranean countries! It includes a wealth of basic info such as languages spoken, religions practiced, the country's capital, its population, as well as other additional facts. 

There is a dropdown feature that allows users to filter by population, language, and independece.

Thank you for interacting with my project!
